# レース結果情報

出走後に得られるレース自体の結果情報。1行のDataFrame。

関数: `DataInterface.get_race_result_info()`

対応元:
- mykeibadb: `RACE_SHOSAI`（`LAP_TIME1`〜`RECORD_KOSHIN_KUBUN`）
- scraping: `ResultPageScraper.get_lap_time()` + `ResultPageScraper.get_corner()`

## 差分B: mykeibadbテーブルとの差分

| カラム | 内容 |
|--------|------|
| ラップ〇m | 新規追加。mykeibadbのLAP_TIME1〜25（200m刻み・0.1秒単位）を100m刻み・秒単位に展開 |
| 前nハロン・後nハロン | 秒単位に変換 |
| nコーナー通過順/周回数 | mykeibadbのCORNER{i}カラムは格納スロット番号と実際のコーナー番号が一致しないが、統一スキーマでは実際のコーナー番号（1〜4コーナー）に対応するカラムに格納する |
