# 馬毎レース情報

出馬表/馬ごとのレース結果/各馬の過去戦績に対応。出馬表の場合は結果に関するカラムの値はNaNに。またscrapingの場合の出馬表ではオッズ情報はNaN。

関数:
- 出馬表: `DataInterface.get_entry()`
- レース結果: `DataInterface.get_result()`
- 過去戦績: `DataInterface.get_past_performances()`

対応元:
- mykeibadb: `UMAGOTO_RACE_JOHO`
- scraping:
  - 出馬表: `EntryPageScraper.get_entry()`
  - レース結果: `ResultPageScraper.get_result()`
  - 過去戦績: `HorsePageScraper.get_past_performances()` + `HorsePageScraper.get_horse_basic_info()`

## 差分A: プロバイダー間の既知差分

### entry/result

| カラム | 内容 | レースコードの例 |
|--------|------|------|
| 調教師/騎手名略称 | 表記ゆれあり | "2023112605050812" |
| 獲得本賞金 | scrapingとmykeibadbで賞金データが若干異なる場合がある | "2023122406050811" |
| 着差コード1 | 同着の場合mykeibadbでは"D__"になる | "2020112309050712" |
| | 失格の場合scrapingでは"3/4+ハナ"のようになる | "2012050605020611" |
| | 降着の場合scrapingではNaNになる | "2012050605020611" |

## 差分B: mykeibadbテーブルとの差分

| カラム | 内容 |
|--------|------|
| 調教師名略称 | 末尾のスペースを除去 |
| 負担重量 | 単位0.1kg → kg（÷10）に変換 |
| 変更前負担重量 | 単位0.1kg → kg（÷10）に変換 |
| 騎手名略称 | 末尾のスペースを除去 |
| 変更前騎手名略称 | 末尾のスペースを除去 |
| 馬体重 | 0（出走取消等で体重未計測）→ NaN |
| 増減符号 | 増減差がNaNまたは0（増減なし）の場合はNaN。出走取消/発走除外/競走除外の馬で増減差=0の場合もNaN |
| 増減差 | 999（計測不能）→ NaN。出走取消/発走除外/競走除外の馬で増減差=0の場合はNaN。DB上NULL（増減なし）→ 0 |
| 異常区分コード | str型に統一。空文字列は"0"に変換 |
| 確定着順 | 0（着順未設定：出走取消/競走除外等）→ NaN |
| 走破タイム | "MSSS"形式 → "M:SS.S"形式に変換。"0000"（競走中止等でタイム未計測）→ NaN |
| 着差コード1/2/3 | DBのスペース文字をアンダーバーに変換。全スペース（"___"=未設定）・空文字列はNaN |
| 1/2/3コーナー順位 | 0（競走中止等でコーナー通過なし）→ NaN |
| 4コーナー順位 | 0 → NaN。新潟芝1000m直線は「走破タイム − 後3ハロン」の昇順ランクで再計算 |
| 単勝オッズ | 単位0.1倍 → 倍（÷10）に変換。0（オッズ未設定：出走取消等）→ NaN |
| 単勝人気順 | 0（未設定：出走取消等）→ NaN。同一オッズは同順位となるよう単勝オッズから再計算 |
| 後4ハロン | 単位0.1秒 → 秒（÷10）に変換。0（計測なし）→ NaN |
| 後3ハロン | 単位0.1秒 → 秒（÷10）に変換。999（計測不能：競走中止等）→ NaN |
| タイム差 | 1/10秒単位の整数 → 秒（÷10）に変換。9999（計測不能：競走中止等）→ NaN |

## 注意事項

※ 異常区分コード
- scrapingで取得できる「除外」は発走除外（code="2"）と競走除外（code="3"）を区別できないため、"3"として扱う。
- scrapingで取得できる「落馬再騎乗」（code="6"）は着順が正常扱いのため区別できず、"0"として扱う。
