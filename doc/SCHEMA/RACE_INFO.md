## 差分A: プロバイダー間の既知差分

| カラム | 内容 | レースコードの例 |
|--------|------|------|
| 曜日コード | mykeibadbでは祝日の場合"3"になる | 2018122306050811 |
| 競走名本題 | 表記ゆれあり | 2023112605050812 |
| グレードコード | scrapingでは特別競走と一般競走を区別できない場合あり | 2025083001020310 |
| コース区分 | scrapingでは障害のコース区分は不明 | 2024122106050710 |
| 本賞金n着 | scrapingとmykeibadbで賞金データが若干異なる場合がある | 2010052305030211 |

## 差分B: mykeibadbテーブルとの差分

| カラム | 内容 |
|--------|------|
| 競走条件コード | 新規追加。5つの年齢別条件カラムの最大値を採用して算出 |
| レース種別 | 新規追加。トラックコードから導出 |
| 芝ダ | 新規追加。トラックコードから導出 |
| 左右 | 新規追加。トラックコードから導出 |
| 内外 | 新規追加。トラックコードから導出 |
| 発走時刻 | "HH:MM"形式に変換 |
