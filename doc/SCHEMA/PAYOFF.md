# 払戻情報

レースの全券種の払い戻し情報。1行のDataFrame。

関数: `DataInterface.get_payoff()`

対応元:
- mykeibadb: `HARAIMODOSHI`
- scraping:
  - 単勝: `ResultPageScraper.get_win_payoff()`
  - 複勝: `ResultPageScraper.get_show_payoff()`
  - 枠連: `ResultPageScraper.get_bracket_payoff()`
  - 馬連: `ResultPageScraper.get_quinella_payoff()`
  - 馬単: `ResultPageScraper.get_exacta_payoff()`
  - 3連複: `ResultPageScraper.get_trio_payoff()`
  - 3連単: `ResultPageScraper.get_trifecta_payoff()`
