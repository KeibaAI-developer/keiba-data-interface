# 単複オッズ情報

単複のオッズ情報。出走頭数分の行数のDataFrame。scrapingの場合はJRAから取得を試みて失敗した場合netkeibaから取得する。

関数: `DataInterface.get_win_show_odds()`

対応元:
- mykeibadb: `ODDS1_TANSHO` + `ODDS1_FUKUSHO`
- scraping:
  - JRA: `scrape_odds_from_jra()`
  - netkeiba: `scrape_odds_from_netkeiba()`

| カラム名 | 型 | scraping | mykeibadb | 説明 | 例 | 差分A | 差分B |
|----------|----|----------|-----------|------|----|------|------|
| レースコード | str | ○ | ○ | 開催年+月日+競馬場コード+回次+日次+レース番号 | "2023112605050812" | | |
| 開催年 | str | ○ | ○ | 該当レース施行年 西暦4桁 yyyy形式 | "2023" | | |
| 開催月日 | str | ○ | ○ | 該当レース施行月日 各2桁 mmdd形式 | "1126" | | |
| 競馬場コード | str | ○ | ○ | 該当レース施行競馬場。コード表2001参照 | "05" | | |

## 差分A: プロバイダー間の既知差分

| カラム | 内容 | レースコードの例 |
|--------|------|------|
| 単勝人気 | 同オッズの場合人気順が上下する場合がある | 2023032506030111 |
| 複勝人気 | 同オッズの場合人気順が上下する場合がある | 2023112605050812 |

## 差分B: mykeibadbテーブルとの差分

| カラム | 内容 |
|--------|------|
| 単勝オッズ | 単位0.1倍 → 倍（÷10）に変換。0（無投票）・取消（`****`）・除外（`----`）はNaN |
| 複勝最低オッズ | 単位0.1倍 → 倍（÷10）に変換。0・取消・除外はNaN |
| 複勝最高オッズ | 単位0.1倍 → 倍（÷10）に変換。0・取消・除外はNaN |
| 単勝人気 | 取消（`**`）・除外（`--`）はNaN |
| 複勝人気 | 取消（`**`）・除外（`--`）はNaN |
